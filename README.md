# 📋 점수제 산정 기준 (Points-Based System)

E-7 비자나 자격 취득을 위한 점수 산정 체계를 바탕으로 합니다.
상세 점수 항목 및 합격 기준은 다음과 같습니다.

## 1. 주요 특징 (Scoring Items)
각 항목은 **기본, 선택, 가점, 감점** 요소로 구성되어 있습니다.

### 🔢 점수 항목 상세
| 구분 | 세부 항목 | 최대 배점 | 비고 |
| :--- | :--- | :---: | :--- |
| **기본항목** | 연간 소득(60), 학력(20), 연령(15), 한국어 능력(20) | **115점** | 필수 평가 요소 |
| **선택항목** | 근속 기간(15), 보유 자산(25), 국내 유학 경험(15) | **55점** | 해당 시 합산 |
| **가점항목** | 국가자격증(20), 추천서(20), 읍·면 지역 근무(10), 사회봉사(5), 납세실적(5) | **60점** | 추가 점수 |
| **감점항목** | 출입국관리법 위반(-5/회), 국내법 위반(-10/회) | **제한 없음** | 누적 감점 |

---

## 2. 합격 기준 (Passing Criteria)

최종 선발을 위해서는 아래의 **두 가지 조건을 동시에 충족**해야 합니다.

> [!IMPORTANT]
> 1. **총점:** 전체 합산 점수 **200점 이상**
> 2. **필수 조건:** 기본항목 점수 합계 **50점 이상**

---

## 🛠️ 활용 방법
1. 본인의 상황에 맞는 항목별 점수를 합산합니다.
2. 기본항목 점수가 50점을 넘는지 먼저 확인합니다.
3. 감점 항목이 있다면 최종 점수에서 차감합니다.
